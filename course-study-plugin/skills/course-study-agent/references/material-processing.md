# Material Processing

## Select the Reader

- PPT/PPTX: extract slide text, titles, notes, diagrams, tables, and sequence; visually inspect slides when layout carries meaning.
- PDF: distinguish digital text from scans; render pages when formulas, diagrams, annotations, or layout matter; use OCR only when needed.
- DOCX: preserve headings, tables, lists, comments, and document order.
- Markdown/TXT: read directly and preserve code blocks, formulas, links, and headings.

Use the environment's dedicated presentation, PDF, or document capability when available.

## Build Grounded Notes

1. Record the exact source path or source identifier.
2. Determine the course, chapter/week, and topic from explicit metadata first.
3. Extract learning objectives, core concepts, definitions, methods, examples, formulas, evidence, and teacher emphasis.
4. Reorganize related ideas into a study-friendly structure.
5. Preserve necessary English terms and discipline-specific notation.
6. Mark unreadable or ambiguous content instead of guessing.

Do not add unrelated encyclopedic material. If model knowledge is helpful, label it as supplementary rather than source-derived.
